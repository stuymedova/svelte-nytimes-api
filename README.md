# Svelte + NY Times API

Svelte Framework + content fetched from The New York Times Top Stories API

## Overview
```
.
└── src
    └── App.svelte
```

## Prerequesites

[Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/)

## Setup

1. Add an [API key](https://developer.nytimes.com/get-started) to `src/App.svelte`
2. Run in terminal:
```
npm install
npm run dev
```
